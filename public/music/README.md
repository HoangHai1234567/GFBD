# Music Files

Thêm file nhạc Happy Birthday instrumental vào đây.

## Hướng dẫn:
1. Tìm file nhạc Happy Birthday instrumental nhẹ nhàng (định dạng .mp3)
2. Đổi tên file thành `happy-birthday.mp3`
3. Copy file vào folder này
4. Nhạc sẽ tự động loop (lặp lại) khi phát

## Gợi ý tìm nhạc miễn phí:
- YouTube Audio Library
- Bensound.com
- Free Music Archive
- Pixabay Music

## Định dạng hỗ trợ:
- .mp3 (khuyến nghị)
- .ogg
- .wav

**Lưu ý:** File phải có tên chính xác là `happy-birthday.mp3`
