# Memory Images

Thêm các ảnh kỷ niệm của bạn vào folder này.

## Hướng dẫn:
1. Thêm ảnh vào folder này (hỗ trợ: .jpg, .jpeg, .png, .gif, .webp)
2. Trang web sẽ tự động load và hiển thị random 6 ảnh
3. Ảnh sẽ tự động thay đổi mỗi 20 giây

## Lưu ý:
- Không giới hạn số lượng ảnh
- Nếu có ít hơn 6 ảnh, sẽ hiển thị tất cả
- Tên file không quan trọng, chỉ cần định dạng đúng
